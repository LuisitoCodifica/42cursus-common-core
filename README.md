# 42cursus Common Core

## libft

Run the following command to download `libft` only:

```bash
bash -c "[[ ! -e libft.zip && ! -d libft ]] \
    && wget -q https://github.com/LuisitoCodifica/42cursus-common-core/releases/download/Projects/libft.zip \
    && unzip libft.zip \
    && rm libft.zip"
```
